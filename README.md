# Content Factory — by Fady George

A powerful content workspace that mimics Claude Projects. Build brand-specific content factories with system prompts, skills, briefing files, and unlimited AI-powered chats.

## Features

- **Multi-project workspace** — Create separate projects for each brand/client with color-coded organization
- **System prompts** — Persistent instructions that frame every chat (brand voice, role, rules)
- **Skills (.skill files)** — Plug in multiple skill playbooks per project, toggle on/off
- **Briefing files** — Drop PDFs, DOCX, TXT, MD, CSV, images as context
- **Chat-specific instructions** — Override or extend project context per chat
- **Bring your own API key** — Works with any AI provider:
  - Anthropic (Claude)
  - OpenAI (GPT-4o, o3)
  - Google (Gemini)
  - DeepSeek
  - Kimi (Moonshot)
  - GLM (Zhipu AI)
  - OpenRouter (400+ models)
  - Any OpenAI-compatible endpoint (Groq, Together, Fireworks, Mistral, Cohere, Ollama, LM Studio)
- **Resizable response area** — Drag the handle between messages and composer
- **Export/Import** — Backup and restore all projects as JSON
- **100% client-side** — Your API key never leaves your browser

## Usage

Just open `index.html` in your browser. No build step, no server, no dependencies required.
